# Razer Keymaps

 Razer Huntsman-TKL and Tartarus Pro Keymaps for anybody who need this.

Layouts for visual repr:

Old:
- <https://www.keyboard-layout-editor.com/#/>
- <https://github.com/ijprest/keyboard-layout-editor>

Forked:

- <https://editor.keyboard-tools.xyz/>
- <https://github.com/adamws/kle-ng>

I'm saving my keymaps and visual representations for each just for easy use when playing after a long pause.

My use-case:

Load (or start anew) JSON from keymap [layout](Doom/Doom_keymap-kb-layout.json) (from any game folder from my repo) at:
- https://editor.keyboard-tools.xyz/

- Modify it for your game key bindings.
- Modify Razer Tartarus pad with such a sheme
- Use 2nd display with PNG keymap exported from website from above.
