# Keymap

## Manoeuvres

| Key | Func | Doc |
| :--: | :-- | :-- |
| A | Attack |  stop to attack any opposing units or base defenses along their route. |
| F | Assault | also attack any enemy buildings along their route. |
| D | Reverse | strategic retreats and baiting tactics.  |
| G | Force | tanks to overrun infantry rather than shoot them. |
| Mouse L+R | FORMATION | combine the expedience of a movement order with the simplicity of the formation preview. |
| Ctrl + X | SCATTER | spread out using this command. |

## Stance
| Key | Func | Doc |
| :--: | :-- | :-- |
| ALT + A | AGGRESSIVE | approach, attack, and pursue enemy units or structures |
| ALT + S | GUARD | approach and attack enemies that enter their line of sight |
| ALT + D | HOLD GROUND | units remain stationary, but fire on any enemies that come within range |
| ALT + F | HOLD FIRE | Units in this stance do not return fire or pursue enemy forces |

## Points and planning

| Key | Func | Doc |
| :--: | :-- | :-- |
| W | Matching Units | Select Matching units |
| Q | All Units | Select All units |
| CTRL + R | PLACE RALLY | point for the currently-selected production queue |
| hold Alt | WAYPOINT | queue up multiple orders by right-clicking |
| CTRL + Z | Planning | toggle -> queue -> toggle to execute the queue, can loop |
